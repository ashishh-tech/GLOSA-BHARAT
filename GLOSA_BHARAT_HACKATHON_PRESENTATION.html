Solution

What is GLOSA?

Green Light Optimal Speed Advisory

An AI system that computes the optimal driving speed so vehicles reach the next signal precisely when it turns GREEN — eliminating unnecessary stops entirely.

AI-Powered Perception

Real-time traffic analysis using YOLOv8 computer vision on existing CCTV infrastructure

Dynamic Speed Advisory

Precise, continuously-updated speed recommendations based on signal phase data

Eco-Friendly Impact

15–20% fuel consumption reduction by eliminating stop-start driving patterns

V2I Communication

Sub-second vehicle-to-infrastructure synchronization via WebSocket protocol

Architecture

System Architecture

Four-layer enterprise design for scalability and real-time performance

01
Perception Layer
YOLOv8 · OpenCV · FastAPI
Real-time visual intelligence analyzing CCTV feeds for vehicle detection and queue density estimation
02
Orchestration Layer
Node.js · Express.js · Redis
Central nervous system for data aggregation, pub/sub messaging, and sub-second routing
03
Advisory Engine
ML Algorithms · Predictive Analytics
Computes optimal speed: Vopt = D / Tgreen with dynamic adjustments for traffic density
04
Interaction Layer
React.js · Leaflet GIS · Socket.io
Digital twin dashboard for traffic authorities and driver HUD interface with live signal countdown
