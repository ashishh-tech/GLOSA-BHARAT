🔌 2. Circuit Wiring Connections

Wire the components exactly as shown below. It will take around 15-20 minutes to assemble.

[LED Connections] Pin 13 ──── 220Ω Resistor ──── RED LED (+) ──── GND Pin 12 ──── 220Ω Resistor ──── YELLOW LED (+) ── GND Pin 11 ──── 220Ω Resistor ──── GREEN LED (+) ─── GND [LCD Display Connections - only 4 wires to I2C] Arduino 5V ──── LCD VCC Arduino GND ──── LCD GND Arduino A4 ──── LCD SDA (Data) Arduino A5 ──── LCD SCL (Clock)
⚠️ LED Polarity Warning: The longer leg of the LED is Positive (+). Connect the resistor to this leg. The shorter leg is Negative (-) and connects directly to the GND line on your breadboard.

🚀 3. Step-by-Step Setup Guide

  1. Assemble Circuit: Place LEDs and LCD on the breadboard and connect jumpers to Arduino pins.
  2. Install Python Libraries: Open terminal and run pip install pyserial requests.
  3. Upload Arduino Code:
    • Open hardware/glosa_hardware/glosa_hardware.ino in Arduino IDE.
    • Go to Tools → Manage Libraries and install LiquidCrystal I2C.
    • Select your Board and Port, then hit Upload.
  4. Start Backend: Run cd backend && npm start in a new terminal.
  5. Start Serial Bridge: Run python hardware/serial_bridge.py (ensure the COM port matches your Arduino).
  6. Start Frontend: Run cd frontend && npm run dev.
